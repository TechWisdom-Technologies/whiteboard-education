# World Class Aid - Partner SOP

Version: March 18, 2026

## 1) Purpose

This SOP guides partner agencies on how to use the Partner Dashboard to submit students, manage documents, and track progress effectively.

## 2) Onboarding SOP

1. Visit /partner and complete agency registration.
2. Upload required NID and supporting documents.
3. Wait for admin review and approval.
4. After approval, log in at /login.
5. Access /partner-dashboard.

Important:
- Pending/rejected statuses are shown at login if access is not yet granted.

## 3) First Login Checklist

1. Open Overview and verify dashboard access.
2. Open My Profile and confirm agency details.
3. Upload profile avatar (optional).
4. Review Notifications module.

## 4) Student Submission SOP

1. Go to /partner-dashboard/students.
2. Click Add Student.
3. Fill mandatory identity and contact fields.
4. Fill academic background (GPA, IELTS, previous qualification).
5. Fill target program details (university/course/intake/degree).
6. Save student profile.

Quality rule:
- Do not submit incomplete student identity data unless unavoidable.

## 5) Document Upload SOP

For each student:
1. Open student detail.
2. Upload required files:
   - Passport
   - Academic transcript
   - IELTS certificate
   - Personal statement
   - Recommendation letter
3. Replace outdated files when newer versions exist.
4. Confirm uploaded file link opens correctly.

## 6) Status Tracking SOP

1. Monitor student status badge in Students and Overview.
2. Read admin notes for each status update.
3. If status is rejected or blocked, review note and resolve gaps.
4. Re-upload corrected documents as needed.

## 7) Notifications SOP

1. Open Notifications frequently.
2. Filter unread first.
3. Mark items read only after action is completed.
4. Use notification context to return to affected student record.

## 8) Marketing Hub SOP

Current module supports:
- Asset listing
- Download action UI
- Referral link copy

Recommended practice:
- Share referral link in campaigns to attribute potential applications.

## 9) Profile Management SOP

1. Keep display name and agency details accurate.
2. Keep contact email active for approval and status communication.
3. Update avatar/profile to improve internal identification.

## 10) Communication SOP

When contacting admin support:
- Include student full name and status
- Include missing-doc issue details
- Include timeline urgency (intake deadline proximity)

## 11) Quality Standards for Partners

- Submit verifiable and legible documents.
- Keep student records truthful and up to date.
- Avoid duplicate student entries.
- Respond quickly to admin notes.

## 12) Common Mistakes to Avoid

- Waiting too long to upload documents.
- Ignoring notification feed.
- Submitting vague or missing target program info.
- Treating rejected status without reading admin notes.

## 13) Weekly Partner Checklist

1. Review all active student statuses.
2. Ensure every student has required document coverage.
3. Follow up on pending admin notes.
4. Update pipeline priorities by nearest intake deadlines.

## 14) Success Metrics for Partner Teams

Track internally:
- Students submitted per month
- Document-complete rate
- Approval progression speed
- Enrollment outcomes

Consistent data quality and timely updates are the fastest way to improve conversion and agency reputation.
